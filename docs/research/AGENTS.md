# AGENTS.md

`docs/research` stores external best practices, comparative analysis, and decision-support notes.

## Owns
- external best-practice notes
- comparative architectural analysis
- critical review notes

## Rules
- Write module-facing docs, examples, and comments in English.
- Prefer primary sources and current official docs when possible.
- Distinguish observed facts from AOR-specific conclusions.
- Keep research concise and actionable for design decisions.
