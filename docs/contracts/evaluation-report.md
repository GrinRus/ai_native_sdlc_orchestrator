# Evaluation report

## Purpose
Suite-based or grader-based report that scores a run, wrapper, route, adapter, or other supported subject.

## Required fields
- `report_id`
- `subject_ref`
- `suite_ref`
- `grader_results`
- `status`
- `evidence_refs`

## Notes
Evaluation reports should preserve thresholds, regressions, and baseline comparisons.
